# Esimvista static website

This package is ready to upload to GitHub as a static site.

## Included
- 100 keyword-targeted eSIM pages
- 50 country pages
- 200 FAQ answers
- Provider hub with official external links
- sitemap.xml
- robots.txt
- CNAME for esimvista.com
- 404 page
- .nojekyll for GitHub Pages compatibility

## GitHub Pages steps
1. Create a new GitHub repository.
2. Upload all files in this folder to the repository root.
3. In GitHub, go to Settings > Pages.
4. Set the source to deploy from the main branch root.
5. If you are using the custom domain, keep the included CNAME file.
6. After DNS is configured for esimvista.com, GitHub Pages will serve the site.

## Notes
- Canonical URLs already use https://esimvista.com/
- Update content over time as provider offers, destination coverage, and device support change.
- Provider links are official brand links, not affiliate links.
